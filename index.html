Inheritance is a mechanism where some CSS properties of a parent element are automatically applied to its child elements.

unset → behaves like inherit if property is naturally inheritable, otherwise like initial.

revert → resets the property to the value it would have from the browser/user stylesheet, ignoring your CSS rules.



The CSS float property is used to position an element to the left or right of its container, allowing text and inline content to wrap around it. It was inspired by print media layouts, such as newspapers and magazines.



RGB defines colors based on the combination of Red, Green, and Blue light. Each component can range from 0 to 255.

RGBA is just RGB with transparency (alpha). Alpha defines opacity: 0 (fully transparent) to 1 (fully opaque).

HEX colors are hexadecimal representations of RGB. Each color component (R, G, B) is represented by two hex digits (00 to FF).

HSL defines colors by hue, saturation, and lightness. Easier to think about in terms of color tones, rather than mixing red/green/blue.

A CSS variable (also called a custom property) is a way to store a value that can be reused throughout your CSS.
